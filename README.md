# Nighthub - Hệ thống xem phim trực tuyến

Nighthub là một nền tảng website xem phim trực tuyến hiện đại với giao diện mượt mà.

Dự án được xây dựng dựa trên **React.js** và **Vite**.

## 📌 Yêu Cầu Hệ Thống (Prerequisites)

Trước khi cài đặt, hãy đảm bảo máy tính của bạn đã cài đặt các phần mềm sau:
1. **Node.js**: (Phiên bản v16.0.0 trở lên). Tải tại: [https://nodejs.org/](https://nodejs.org/)
2. **Git**: Dùng để tải mã nguồn. Tải tại: [https://git-scm.com/](https://git-scm.com/)

---

## 🚀 Hướng Dẫn Cài Đặt & Chạy Dự Án (Local Development)

Để chạy dự án này trên máy cá nhân của bạn, vui lòng làm theo các bước dưới đây:

### Bước 1: Clone dự án về máy
Mở Terminal (hoặc Command Prompt / Git Bash) và chạy lệnh sau để tải mã nguồn về máy:

```bash
git clone https://github.com/Len-Kaito/NightHub.git
```

### Bước 2: Di chuyển vào thư mục dự án
```bash
cd nighthub-react
```

### Bước 3: Cài đặt thư viện (Dependencies)
Nighthub sử dụng `npm` làm trình quản lý gói. Chạy lệnh sau để cài đặt các gói cần thiết:
```bash
npm install
```

### Bước 4: Khởi chạy Server Development
Sau khi quá trình cài đặt hoàn tất, hãy khởi động server dành cho nhà phát triển bằng lệnh:
```bash
npm run dev
```

### Bước 5: Truy cập website trên trình duyệt
Sau khi chạy lệnh ở Bước 4, Terminal sẽ hiển thị một đường dẫn cục bộ (thường là `http://localhost:5173/`).
- Bạn có thể **Copy** đường dẫn đó và **Dán (Paste)** vào thanh địa chỉ của trình duyệt web (Chrome, Edge, Safari...).
- Hoặc giữ phím `Ctrl` (trên Windows) / `Cmd` (trên Mac) và click trực tiếp vào đường link trong Terminal để mở tự động.
